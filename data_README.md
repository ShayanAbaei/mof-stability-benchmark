# Data provenance

This repository does not redistribute the raw benchmark input files.

The workflow used the public CoRE MOF 2024 dataset release together with the following five CSV files:

- `ASR_data_SI_20250204.csv`
- `FSR_data_SI_20250204.csv`
- `ION_data_SI_20250204.csv`
- `ASR_FSR_check.csv`
- `12089-recommended-screening-list.csv`

Users should download these files from the official Zenodo record and place them in a local folder named `raw_data/` before running the benchmark notebooks.
